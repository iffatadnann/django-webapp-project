# Django Homepage Project

This is a Django web application built as a learning project.

## Features

- Django based web application
- Homepage app with views and templates
- Static files support
- SQLite database
- Basic project structure following Django conventions

## Project Structure

MYDJANGOPROJECT
│
├── homepage (Django app)
├── myproject (Project settings)
├── templates
├── static
├── manage.py
├── requirements.txt

## Installation

1. Clone the repository

git clone https://github.com/yourusername/django-homepage-project.git

2. Navigate to the project

cd django-homepage-project

3. Create virtual environment

python -m venv venv

4. Activate environment

Windows:
venv\Scripts\activate

5. Install dependencies

pip install -r requirements.txt

6. Run migrations

python manage.py migrate

7. Run server

python manage.py runserver
